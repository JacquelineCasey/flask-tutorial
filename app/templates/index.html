This is the homepage.

This site is used for testing various Heroku and Flask features that might make it into other projects. Navigate to some other pages on the site to see various features.

Link Purpose
This Page To show off how a page might use HTML and link to other pages. Also finds and uses a very simple CSS stylesheet.
eflask-app-067.herokuapp.com/sample A Page to link to, which also uses HTML.
eflask-app-067.herokuapp.com/deprecated Another page to link to. This one is coded slightly differently, sending text directly instead of rendering an HTML file.
eflask-app-067.herokuapp.com/environment This page asks the app to read an environment variable, and it says something different depending on if this is running locally (on my macbook), or on Heroku. This is good for hiding something (ie a password or a database key) from prying eyes.
eflask-app-067.herokuapp.com/redirect Accessing this link triggers the app to send a redirect message instead of an actual webpage. It redirects to the deprecated homepage.
eflask-app-067.herokuapp.com/secret This page is hidden behind a login system, so if you are not logged in, you will be redirected to the login page. The system is set up so it remembers where you wanted to go to after login, however.
eflask-app-067.herokuapp.com/login This is the login page. Accessing it through this link will cause it to send you back here, but you will still be logged in. Try heading to the secret page afterwards. PS: The password is "testpassword".
eflask-app-067.herokuapp.com/logout Currently, this route triggers an immediate logout, but it immediately redirects you here. Try accessing the secret page again; you will need to log in to see it.
